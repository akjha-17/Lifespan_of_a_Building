# Lifespan_of_a_Building

To predict lifetime of a building that is to find out whether there is any chance of a 
structural collapse by training a machine learning based model. Also to depict the 
correlations using various plotting techniques. This will help us identify if a building is safe 
or if it will fail catastrophically resulting in a collapse
